# VOLTab <br>
Firefox/Waterfox addon to adjust the volume of html media (bandcamp, soundcloud, youtube, etc) per-tab or window.

Light-weight, non-intrusive. Simple.<br>
Works for Firefox, Waterfox, etc.<br>
Cellphones untested.<br>
